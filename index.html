Как разместить на GitHub Pages

Разместите этот сайт бесплатно через GitHub Pages за несколько минут.

1
Создайте репозиторий на github.com — назовите его имя-проекта или username.github.io
2
Загрузите файл index.html в корень репозитория через кнопку «Add file» → «Upload files»
3
Зайдите в Settings → Pages, в разделе Source выберите ветку main и папку / (root)
4
Нажмите Save — через 1-2 минуты сайт будет доступен по адресу username.github.io/имя-репозитория
5
Для обновлений просто заменяйте index.html в репозитории — GitHub сам пересоберёт сайт
